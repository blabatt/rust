\subsection*{Modules}
\code{mod fn my\_function1()\{\};} // private \\
\code{pub mod fn my\_function2()\{\};} // public \\
// Module in another file, still need to \say{use}: \\
\code{mod file\_name;} \\
\subsection*{Paths}
\code{use absolute::path::to::module;} \\
\code{use relative\_path::to::\{mod1, mod2\};} \\
\code{use path::to::\{sub\_path::mod1, mod2\};} \\
\code{use std::io::Result as IoResult;} // aliasing \\
\code{pub use rand::Rng;} // \say{re-exporting} \\

\subsection*{Crates}
<crate> = <lib\_crate> | <bin\_crate> \\

\subsection*{Packages}
Packages contain ($\ni$) crates, according to:
\begin{itemize}[label={}]
\item $\forall$ packages $\ni$ <crate>+ 
\item $\forall$ packages $\ni$ <lib\_crate>? 
\item $\forall$ packages $\ni$ <bin\_crate>* 
\end{itemize} \
